# pytracestats
A simple python script to plot the bandwidth of some huge traces

# usage
```
usage: bandwidth_extract.py [-h] [--interval INTERVAL] [--stop-after STOP]
                            [--output OUTPUT]
                            filename [filename ...]

positional arguments:
  filename

optional arguments:
  -h, --help           show this help message and exit
  --interval INTERVAL
  --stop-after STOP
  --output OUTPUT
```
